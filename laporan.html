Laporan: Proses Pengerjaan Tugas Metaflow

Spesifikasi Laptop/Komputer yang Digunakan

  • Laptop: MacBook Air M2
  • CPU: Apple M2 chip
  • RAM: 8GB
  • Storage: 256GB

Kendala yang Dihadapi

  1. Minimnya Tutorial untuk MacBook:

    Sulit menemukan tutorial yang sesuai untuk penggunaan Metaflow dan alat terkait di macOS. Beberapa tutorial berfokus pada penggunaan di Windows atau Linux, yang menyebabkan kebingungan saat mencoba menyesuaikan dengan macOS.

  2. Pemilihan Grup WhatsApp:

    Dalam konteks tugas, ada kebingungan mengenai pemilihan grup WhatsApp sebagai salah satu sumber dataset, yang memengaruhi pemilihan file dataset yang digunakan.

  3. Kebinguan File Dataset (.tar atau .csv?):

    Tidak jelas apakah file dataset yang harus digunakan adalah dalam format .tar atau .csv. Ketidakjelasan ini menyebabkan kebingungan dalam memulai proses analisis data.

  4. Masalah pada Proses Metaflow:
    • Gagal Install Metaflow: Mengalami kesulitan saat menginstal Metaflow di macOS, terutama dengan ketidakcocokan versi Python atau ketergantungan lainnya.
    • Gagal Eksekusi Metaflow: Saat menjalankan skrip Metaflow, terdapat masalah pada beberapa tahap, seperti error saat membaca file atau kesalahan dalam konfigurasi parameter yang tidak dikenali.

Langkah yang Sudah Dicoba

  • Menginstal Metaflow menggunakan pip install dan mencoba menjalankan beberapa langkah dalam skrip, meskipun mengalami kendala terkait ketergantungan.
  • Menggunakan dataset dari file .csv yang telah dibersihkan, tetapi terkendala dalam pemrosesan data, terutama dalam hal memahami struktur dataset.

Solusi yang Dicoba

  • Menggunakan dokumentasi dari GitHub Metaflow untuk mencoba mencari solusi terhadap masalah instalasi.
  • Membaca dan mencoba menjalankan skrip di beberapa IDE (misalnya VSCode dan PyCharm) untuk mencari cara yang lebih stabil.
  • Mengonversi file .tar ke .csv untuk memastikan format yang digunakan sesuai dengan kebutuhan analisis.

Rencana Ke Depan

  • Menyelesaikan proses instalasi dan eksekusi Metaflow pada macOS dengan mengikuti lebih banyak tutorial atau forum diskusi terkait Metaflow di macOS.
  • Memastikan format dataset yang digunakan adalah .csv, karena ini lebih mudah diproses dalam analisis data.
  • Menyelesaikan langkah-langkah analisis clustering dengan Metaflow dan memastikan output yang dihasilkan dapat diinterpretasikan dengan baik.
